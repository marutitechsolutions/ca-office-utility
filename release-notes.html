v1.1.2 Latest
April 27, 2026

The High-Precision Accounting & Multi-Platform Update. This version introduces macOS support and a refined accounting consensus engine for complex bulk invoice parsing.

Accounting & Extraction
  • Multi-Invoice Parsing — Enhanced reliability for bulk PDFs containing hundreds of unique invoices, ensuring stable extraction across all pages.
  • Anchored Consensus Engine — Advanced math logic that uses label proximity (BASIC, DISC) to calculate taxable bases with 100% precision.
  • Noise Header Filtering — Automated exclusion of phone numbers, PIN codes, and other non-accounting numeric noise from extraction candidates.
New Platform & UI
  • Official macOS Build — Desktop application now fully supported on macOS, optimized for both Intel and Apple Silicon Macs.
  • Fixed a UI issue in CMA / DPR Project Finance tab where the 'Copy from Req.' button overlapped the input label.
Patched glassmorphism rendering bugs on older browsers.

v1.1.1 Security & Logic Fix
April 17, 2026

Critical stability update fixing CMA report generation crashes and introducing a decentralized Peer-to-Peer (P2P) licensing system for improved LAN reliability.

CMA / DPR Builder Fixes
  • Report Generation Crash — Fixed `AttributeError: provisions` when generating Word/PDF reports for projects with historical audited data.
  • Robust Attribute Access — Legacy project files now load safely with enhanced fallback logic for audited financial fields.
New Licensing Architecture
  • P2P Floating Seats — Introduced a decentralized Peer-to-Peer licensing model. Any PC on the LAN can now host seat tracking independently.
  • No Server Needed — Restricted licensing functions no longer require a specific "Server" PC to be started first.
  • Network Seat Insights — New "Office License Server" view showing real-time shared seat usage across all network peers.

v1.0.0 Stable
April 14, 2026

First production release of CA Office PDF Utility — a secure, offline desktop automation suite built for Chartered Accountants, tax consultants, and audit professionals.

Core Modules Included
  • PDF Engineering — Merge, split, reorder, compress, delete pages, add password protection, and manage PDFs with high-fidelity document preservation.
  • Smart Invoice Parser — AI/OCR-powered extraction of invoice numbers, dates, GSTIN, tax values, and totals into structured Excel reports. Handles diverse invoice formats.
  • CMA / DPR Builder — Generate banker-ready CMA Data and Detailed Project Reports with financial projections, schedules, ratio analysis, and professional formatting.
  • Digital Signature — Bulk document signing using DSC tokens with configurable stamp positioning, visual signature placement, and secure batch workflow.
Key Features
  • 100% offline processing — no files uploaded to any server
  • Professional dark-themed desktop interface built with modern UI framework
  • Windows 10 and Windows 11 compatible
  • Secure local installation — no admin rights required
  • Optimized for daily CA office document workflows
